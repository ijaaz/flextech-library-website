Arduino Tutorial 1: Introduction to Arduino

Welcome to the FlexTech Library's introductory Arduino tutorial. The intention of this tutorial is to familiarize students with the basics of an Arduino Uno Microcontroller.

Required Materials:

  • Arduino Uno Microcontroller
  • USB A/B Cable
  • A Functioning Computer with a USB A port.
  • Arduino Software

Hardware

Let's take a minute to get ourselves familiar with the hardware. The Arduino Uno Microcontroller is a small function machine that is designed to retain and execute a code, and read and write to and from different inputs and outputs. The physical board has many pins on it, where you can plug in either inputs or outputs. It also has a USB B port, for use with a computer, and a power port, which is used when you need to power the arduino without a computer. On one side of the board, there are pins labeled with numbers 0 - 13; these are the input/output pins. Pins labeled 'GND' are the ground for a power cable.
